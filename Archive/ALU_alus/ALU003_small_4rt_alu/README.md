Based on Aminotreal's 3rt CCA. 9*10 in size. Fixed some quirks and the truth table from the original message.
## Input
A and B (8 bit), !A, !B, Xor (FC), And, Cin, Rsh
## Output
C (8 bit), Cout
## Speed
4rt (For all operations)
## Notes
Greyed areas mean only one input should be toggled, but either works.
