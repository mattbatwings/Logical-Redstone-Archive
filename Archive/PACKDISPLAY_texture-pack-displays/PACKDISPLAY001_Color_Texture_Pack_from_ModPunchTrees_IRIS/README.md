# Color Texture Pack (from ModPunchTree's IRIS)
**Authors:** *ModPunchTree*

**Endorsed by:** *mattbatwings*

**Tags:** *Tested & Functional, Recommended*

**Original post:** [View on Discord](https://discord.com/channels/1361854816837435433/1472412352136872029)

Adds 16 different color faces to redstone dots based on their signal strength (Right click redstone dust). Great for a 2x2 pixel color screen.
## Input
redstone signal strength
## Output
16 different color faces on redstone dots
