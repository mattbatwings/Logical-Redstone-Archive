256 Bytes of cached serial RAM (6 bpb), including a 32 Byte 1-line cache.
## Input
5b address within cache, input data, page addr for swap
## Output
Data at address
## Speed
10 hz swap, 9t read
## Notes
Read cannot be on while swapping.
