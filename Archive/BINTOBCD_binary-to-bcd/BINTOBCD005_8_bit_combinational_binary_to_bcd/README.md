A nice combinational 8 bit binary to BCD using a compact ripple carry adder (by Ammorth) with a 1 wide gap
## Input
8b
## Output
3 digit bcd
## Speed
50 rt
## Notes
Due to its inherent design, this device is unable to synchronize.
