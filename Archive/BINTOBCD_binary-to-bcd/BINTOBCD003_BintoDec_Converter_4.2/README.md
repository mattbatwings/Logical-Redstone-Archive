Converts a Binary number into a Decimal to tile with 7 segment displays.
## Input
Nb number
## Output
N digit decimal number (signal strength)
## Speed
2*b + 1 ticks (b = number of bits)
## Notes
Infinitely Expandable (7x4x2 per cell), Uses TTP to remove directionality issues.
