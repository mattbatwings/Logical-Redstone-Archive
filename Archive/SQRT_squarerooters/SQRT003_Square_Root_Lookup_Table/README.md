A lookup table preprogrammed to calculate the square root of an unsigned integer
## Input
8 bit value
## Output
8 bit value - unsigned 4.4 fixed point
## Speed
9 ticks
## Notes
If you're looking for a much smaller square root design, then this should not be your primary choice. However, this square root is very fast.
