Seven very sleek (4-wide) single bus IO-ports with a total volume of 3915 Blocks.
## Input
7 8b binary Inputs
## Output
7 8b binary Outputs
## Speed
3rt Read, 6rt Write
## Notes
Control-/Feedback-lines (top to bottom): Data Out, Toggle Write (OFF = Read, ON = Write), Address bit 2, Address bit 1, Address bit 0
