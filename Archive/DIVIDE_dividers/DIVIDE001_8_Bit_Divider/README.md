An 8 bit combinational divider
## Input
2 8 bit levers for A/B
## Output
2 8 bit outputs for quotient/remainder
## Speed
52 ticks for quotient, 55 ticks for remainder (pipelined at 5Hz)
## Notes
Uses Aminotreal's 8 bit 5Hz CCA, optimised for throughput. Anything /0 will result in the quotient being 255
