A lookup table preprogrammed to calculate the sine of an angle (given in radians). Angle range 0 to 255/128 (1.992) radians.
## Input
8 bit angle - unsigned 1.7 fixed point, radians
## Output
8 bit value - signed 1.7 fixed point
## Speed
9 ticks
## Notes
Angle is in RADIANS!
