MCHPRS compatible "random" number generator (16 long): Linear Feedback Shift Register
## Input
Randomizer pulse
## Output
8 bit random number
## Speed
6t randomization + reset time
## Notes
Built in pulse shortener. You can also change the randomizer seed by changing the states of the locked repeaters
