A decoder that does not use torches, vertical to horizontal (2 ws, staggered). Part of the decoder can also be removed in order to get the inverted output.
## Input
Up to 8 bit binary value (Schematic: 3 bits)
## Output
One-hot binary value, expandable (Schematic: 8 outputs)
## Speed
1 tick inverted output, 2 ticks output (for 8 outputs)
