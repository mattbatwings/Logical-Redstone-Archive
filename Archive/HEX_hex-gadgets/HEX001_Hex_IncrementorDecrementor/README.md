# Hex Incrementor/Decrementor
<img alt="hex_incdec.png" src="images/hex_incdec.png?raw=1" height="300px">

**Tags:** *Tested & Functional, Recommended*

**Original post:** [View on Discord](https://discord.com/channels/1361854816837435433/1472409734542921728)

Increments or decrements a hex number.
## Input
Increment button, Decrement button
## Output
Hex number
## Speed
2 redstone ticks per inc/dec
## Notes
You can increase/decrease by more than one by increasing the pulse length.
