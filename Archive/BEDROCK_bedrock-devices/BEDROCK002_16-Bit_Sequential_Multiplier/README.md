# 16-Bit Sequential Multiplier
**Authors:** *Zenith*

**Endorsed by:** *mattbatwings*

**Tags:** *Tested & Functional*

**Original post:** [View on Discord](https://discord.com/channels/1361854816837435433/1473732247503900775)

A compact and fast sequential 16 bit binary multiplier for Minecraft Bedrock Edition
## Input
16-Bit No. (A, B), reset button, and calculate button
## Output
32-Bit No.
## Speed
Dependent on A, B inputs but longest case is 93 RT
## Notes
Uses pistons to reset (Not Solid State), and only works in Bedrock Edition since Java QC breaks it.
