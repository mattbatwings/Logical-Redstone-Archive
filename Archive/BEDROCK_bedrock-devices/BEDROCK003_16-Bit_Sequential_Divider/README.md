A nice sequential 16 bit binary divider for Minecraft Bedrock Edition
## Input
16-Bit No. (A, B), reset button, and calculate button
## Output
16-Bit No. (Answer & Remainder)
## Speed
213 RT
## Notes
Uses pistons to reset (Not Solid State), and only works in Bedrock Edition since Java QC breaks it. The bigger number goes in the left input while the smaller number goes in the right input.
