A nice, and compact 16-Bit BCD to Binary converter for Minecraft Bedrock Edition
## Input
BCD Digit, reset button, and calculate button
## Output
16-Bit No.
## Notes
- BCD Digits must be inputted one after the other (Input BCD Digit, press the calculate button, and repeat)

- Uses pistons to reset (Not Solid State), and only works in Bedrock Edition since Java QC breaks.

- Highest BCD No. this can convert is 65535
